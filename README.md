# 수도꼭지 일기 (The Tap Dairy)

자주 우는 사람의 얼만큼 왜 울었는지 기록하는 눈물 일기

A tear diary of someone nicknamed Tap

## Demo

You can access the live demo of this project at [수도꼭지 일기(The Tap Dairy)](https://the-tap-diary.vercel.app/).
